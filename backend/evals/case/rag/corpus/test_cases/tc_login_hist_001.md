# 历史测试用例：用户登录模块

## 场景：用户登录

### TC-LOGIN-HIST-001 用户名密码错误提示

- **优先级**：P0
- **前置条件**：用户已打开登录页
- **步骤**：
  1. 输入正确用户名、错误密码
  2. 点击登录
- **预期**：页面提示「用户名或密码错误」，不暴露具体哪一项错误

### TC-LOGIN-HIST-002 验证码过期刷新

- **优先级**：P1
- **前置条件**：验证码已展示超过 5 分钟
- **步骤**：
  1. 输入任意用户名密码与过期验证码
  2. 点击登录
- **预期**：提示验证码过期，自动刷新验证码图片
